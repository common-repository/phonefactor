Adds two-factor security to the WordPress login system.

== Description ==

PhoneFactor adds two-factor security to the WordPress login system by requiring users to confirm a login attempt by accepting a call from PhoneFactor and pressing the # key.

== Installation == 

1. Download the plugin archive and expand it (you've likely already done this).
2. Upload the phonefactor directory to your wp-content/plugins directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for PhoneFactor.
4. Click the 'register' link in the banner and submit the registration form.
5. If successful, navigate to your Profile page and enter your telephone number.
5. Once your PhoneFactor number is entered click the 'Test' button, accepted the call from PhoneFactor and press the # key.
6. Congratulations, you've just installed and setup PhoneFactor.
